## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/gollavinaykumar/Learnings.git
   cd Learnings
   ```

2. **Browse the materials** - Open any README.md file in folders to get an overview

3. **Read sequentially** - Start with numbered concept files (01_, 02_, etc.)

4. **Practice** - Use interview questions and coding examples to test your knowledge
