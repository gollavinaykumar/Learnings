# 📖 What is Netfilter?

Netfilter is a framework inside the Linux kernel.

It allows the kernel to inspect, modify, allow, reject, or redirect packets while they travel through the networking stack.

Think of it as:

```
Packet

↓

Checkpoint

↓

Decision

↓

Continue
```

---

# Why Does Linux Need Netfilter?

Suppose a packet arrives:

```
Internet

↓

Linux
```

Linux may need to:

✔ Block it

✔ Accept it

✔ Change its IP

✔ Forward it

✔ Redirect it

Without Netfilter,

none of this would be possible.

# PREROUTING

This is the first stop.


# Interview Questions

## What is Netfilter?

Netfilter is a Linux kernel framework that inspects, filters, modifies, and forwards network packets.

---

## What is a Netfilter Hook?

A hook is a predefined point in the kernel networking stack where packets can be examined or modified.

---

## Name the five Netfilter hooks.

- PREROUTING
- INPUT
- FORWARD
- OUTPUT
- POSTROUTING

---

## Where does DNAT usually occur?

PREROUTING.

---

## Where does SNAT usually occur?

POSTROUTING.

---

## What is the difference between Netfilter and iptables?

Netfilter is the kernel framework.

iptables is a userspace tool used to configure rules for Netfilter.

---

## Why does Docker depend on Netfilter?

Docker uses Netfilter for NAT, port publishing, packet forwarding, and firewall rule management.

# Summary

Netfilter is the packet-processing engine inside the Linux kernel.

```
Incoming

↓

PREROUTING

↓

Routing

↓

INPUT / FORWARD

↓

POSTROUTING

↓

Outgoing
```

Key concepts:

- Netfilter processes packets at well-defined hook points.
- PREROUTING sees packets before routing.
- INPUT handles packets destined for the local machine.
- FORWARD handles packets passing through the machine.
- OUTPUT handles locally generated packets.
- POSTROUTING processes packets just before transmission.
- Docker, Kubernetes, NAT, and firewalls all rely on Netfilter.

Understanding Netfilter prepares you for **iptables**, where you'll learn how administrators and Docker actually install rules into these hooks.
